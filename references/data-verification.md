# 信息渠道与数据核验

## 志愿填报最需要的五类数据

1. 一分一段表：把分数转换为全省位次。
2. 招生计划：确认学校、专业组、专业、人数、学费、校区、选科要求。
3. 历年投档数据：判断能不能进学校或专业组。
4. 专业录取数据：判断能不能进具体专业，尤其老高考或专业分流规则明确的省份。
5. 招生章程：确认录取规则、调剂规则、体检限制、专业级差、单科要求。

## 资料可信度分层

- 第一层：省教育考试院、官方招生计划、院校招生章程、阳光高考。
- 第二层：学校本科招生网、学院官网培养方案、学校就业质量报告。
- 第三层：历年录取数据库、志愿填报工具、第三方统计表。
- 第四层：学长学姐经验、公开经验分享、论坛问答、校园社群、校友反馈等非官方材料。

Skill 使用时应先用第一层确认规则，再用第二三层做判断，第四层只做生活体验和风险线索。

## 需要强制二次核验的内容

- 2026 年新增专业、招生计划变化、专业组调整。
- 投档规则、批次合并、调剂政策变化。
- 医学、公安、军校、航海、飞行、定向、公费师范等特殊要求。
- 中外合作、联合培养、校区搬迁、住宿条件。
- 保研率、就业率、薪酬、升学率。
